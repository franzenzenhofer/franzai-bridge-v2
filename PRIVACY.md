# Privacy Policy - FranzAI Bridge V2

**Last updated: February 7, 2026**

## Data Collection

FranzAI Bridge V2 does **not** collect, transmit, or store any personal data. All extension data (API keys, settings, request logs) is stored locally on your device using Chrome's storage API.

## What Data the Extension Accesses

- **API Keys**: User-configured API keys are stored locally and injected into outgoing API requests. Keys never leave your device except as part of the API requests you initiate.
- **Request Logs**: The extension logs API request metadata (URLs, status codes, timing) locally for the sidepanel inspector. Logs are stored only in your browser and can be cleared at any time.
- **Settings**: User preferences (enabled domains, logging preferences) are stored locally using Chrome storage.

## Data Sharing

This extension does **not**:
- Collect or transmit any personal information
- Use analytics or tracking services
- Share data with third parties
- Store any data on external servers

## Permissions

- **storage**: Store settings and request logs locally
- **sidePanel**: Display the request inspector panel
- **activeTab**: Identify which tab initiated API requests
- **identity**: Optional Google OAuth for API authentication
- **host_permissions (all URLs)**: Route API requests through the extension to bypass CORS
